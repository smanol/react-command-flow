# React Command Flow

A React component for multi-step command sessions.

## Installation

```bash
npm install
```

## Usage

```jsx
import React from "react";
import CommandLine from "react-command-flow";

const App = () => (
  <div>
    <CommandLine />
  </div>
);

export default App;
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```
